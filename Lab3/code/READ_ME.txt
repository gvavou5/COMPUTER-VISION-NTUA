Για να μπορέσετε να τρέξετε τον κώδικα της άσκησης θα πρέπει να κάνετε τις παρακάτω ενέργειες:

1) Όλα τα αρχεία που είναι στο φάκελο code του zip αρχείου θα πρέπει να είναι μέσα σε φάκελο ο οποίος περιέχει το φάκελο samples(μαζί με το ότι περιέχει), ο οποίος μας δίνεται στο material της άσκησης.

2) Όλα τα .m files τα οποία υπάρχουν στο φάκελο code θα πρέπει να είναι όλα μαζί στο ίδιο directory για να τρέξει ο κώδικας σωστά.

Αν κάνετε όλα τα παραπάνω θα μπορείτε να τρέξετε τον κώδικα.
